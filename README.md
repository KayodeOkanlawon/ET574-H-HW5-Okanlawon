# Student Performance Visualizer

## Project Overview
This application visualizes student performance data using Python and a graphical user interface built with wxPython.

The program loads a dataset and displays a scatter plot comparing study time and final grades.

## Features
- GUI application using wxPython
- CSV dataset loading using Pandas
- Scatter plot visualization using Matplotlib

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- wxPython

## Setup Instructions

Install the required libraries:

```bash
python3 -m pip install wxPython pandas matplotlib numpy
```

## Usage Details

Run the application using:

```bash
python3 App.py
```

Click the button to load the student dataset and display the graph.

## Dataset Information

Dataset:
UCI Student Performance Dataset

https://archive.ics.uci.edu/dataset/320/student+performance

The dataset contains information about student academic performance, study time, and final grades.
